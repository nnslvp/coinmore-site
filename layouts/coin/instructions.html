lolMiner

  1. Download last release of lolMiner and unzip files
  2. Update alph file, for example: dual_mine_eth_aleph.bat 
                        
                      set "ALEPHPOOL=detect-my-region.alephium-pool.com:20032"
                      set "ALEPHWALLET=your_wallet"

  3. Run alph file, for example: dual_mine_eth_aleph.bat
  4. PROFIT!

Question 1

Alephium mining pool is a mining pool configured in such a way that each miner works independently of the others. The block reward goes to only the miner who found it, others do not get anything. Block search time depends on your hashrate and luck.
